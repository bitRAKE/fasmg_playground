Collection of functions in x86 which generate sequences listed in:

The On-Line Encyclopedia of Integer Sequences® (OEIS®)
https://oeis.org/


Conventions:

Each file is named according to the sequence number and script type:
	*.asm		x86-64
	*.fasmg		https://flatassembler.net/docs.php?article=fasmg

Several algorithms may be included in each file to expose sufficient novelty.
